# گزارش بررسی و اصلاح کدهای MedNegotiator

## جمع بندی
نسخه اولیه بیشتر یک دمو مینیمال بود و با نسخه توصیف شده در مقاله انطباق کامل نداشت. مهم ترین شکاف ها در منطق مذاکره، نگهبان سازگاری، پیکربندی مدل، و فایل وابستگی ها دیده شد.

## ایرادهای اصلی نسخه اولیه

1. `requirements.txt` خراب بود و به شکل یک خط به هم چسبیده ذخیره شده بود؛ بنابراین نصب وابستگی ها عملا شکست می خورد.
2. در `main.py` کلید API به صورت هاردکد شده و نمایشی قرار داشت و از `.env` استفاده نمی شد.
3. مدل پیش فرض با مقاله هم خوانی نداشت؛ مقاله از `GPT-4o` با دمای پایین استفاده کرده، ولی پروژه از الگوی قدیمی استفاده می کرد.
4. در `protocol.py` متغیر `current_proposal` عملا به روز نمی شد؛ بنابراین پروتکل alternating-offers درست اجرا نمی شد.
5. منطق پذیرش فقط یک شرط ساده روی `accepted_bool` و Nash داشت و Safety Guardrails مقاله را پیاده نمی کرد.
6. `engine.py` نسخه خیلی ساده و ناقصی از تابع مطلوبیت فنی داشت و قیود سخت را به شکل درست مدل نمی کرد.
7. Consistency Guard و Semantic Anchoring که در مقاله جزو اجزای اصلی هستند، در کد وجود نداشتند.
8. Mediator در نسخه اولیه فقط یک فراخوانی متنی بود و بهینه سازی جمعیتی/کاندیدامحور نداشت.
9. پروژه بدون API key عملا قابل اجرای قابل اتکا نبود.
10. README هم چند خطای فرمت و نصب داشت.

## اصلاحاتی که اعمال شد

1. فایل `requirements.txt` بازنویسی شد.
2. بارگذاری تنظیمات از `.env` اضافه شد.
3. مدل پیش فرض به `gpt-4o` تغییر کرد.
4. موتور مطلوبیت بازنویسی شد تا `U_clin`، `U_tech` و Nash Product به مقاله نزدیک تر شوند.
5. `ConsistencyGuard` اضافه شد.
6. Semantic Anchoring به صورت سبک و قابل اجرا بدون سرویس embedding اضافه شد.
7. پروتکل مذاکره بازنویسی شد تا round-based، alternating-offers، deadlock detection و mediator fallback داشته باشد.
8. Mediator به صورت candidate-based compromise generator پیاده سازی شد.
9. fallback قطعی برای حالت بدون API key اضافه شد.
10. README و CHANGELOG به روز شدند.

## نتیجه تست

نسخه اصلاح شده کامپایل شد و در سناریوی نمونه از مقاله/کد:

- پیشنهاد اولیه با `1 ms latency` رد شد.
- ضدپیشنهاد فنی با `200 ms + edge caching + async analytics + tiered storage + audit/encryption/override` پذیرفته شد.
- خروجی نهایی به توافق رسید (`AGREED`).
